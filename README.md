# Dartboard Checkout Trainer


Interactive web app for practicing all 3-dart finishes up to 170.


## Features
- Automatically generates all valid 3-dart checkouts from 170 → 2
- Clickable SVG dartboard
- Multiplier selection: Single, Double, Triple, Bull
- Validates guesses (sum equals target, final dart is double)
- Tracks streaks, best streak, and history
- Export/import progress and print checkouts


## Usage
1. Clone or download this repository
2. Open `index.html` in a modern browser
3. Click dartboard segments to practice finishes


## Deployment (GitHub Pages)
1. Push files to GitHub repository
2. Go to **Settings → Pages**
3. Enable Pages from main branch, root folder
4. Access site at `https://<username>.github.io/<repo>/`
